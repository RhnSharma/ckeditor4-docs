Pasting content from Microsoft Word Documentation

Overview

The Paste from Word feature is provided through the Paste from Word plugin. It was heavily reworked in CKEditor 4.6 and by default is available in the Standard and Full distributions.

This feature allows you to paste content from Microsoft Word and maintain original structure and formatting. After creating a document in Word you can simply copy it to CKEditor 4 and retain images, font styles, heading levels, nested lists, complex tables and other advanced features.

When the plugin is enabled, it automatically detects Word content and transforms its structure and formatting to clean, semantic HTML. It also adds the   toolbar button which makes it possible to paste clipboard data this way only on demand.

Pasting content without filtering

To test how Paste from Word works, download the sample Word document with complicated structure and styling, open it in Microsoft Word, copy the content, and paste it into CKEditor 4. Check the clean HTML source code generated by CKEditor 4 using the Source button.

Pasted content may be filtered by additional Advanced Content Filter rules - see the second sample below. The content appearance in CKEditor 4 also depends on styling used in contents.css files.

Filtering pasted content with ACF
